# To-Do App

## Description
A simple and interactive To-Do list application built using **HTML, CSS, and JavaScript**.  
Users can **add, delete, and mark tasks as completed**. This project helps demonstrate basic DOM manipulation, event handling, and responsive UI design.

## Features
- Add new tasks.
- Delete tasks individually.
- Mark tasks as completed.
- Interactive and responsive design.

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Open `index.html` in your browser.
2. Type a task in the input box and click **Add Task**.
3. Click on a task to mark it as done.
4. Click **Delete** to remove a task.
